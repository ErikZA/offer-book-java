## Fluxo recomendado

1. Validar Docker: `./init.ps1` (PowerShell) ou equivalente no seu ambiente.
2. Rodar testes: `.\scripts\build.ps1 docker-test` (Windows) ou `make docker-test` (Linux/macOS).
3. Consultar o guia canônico e, quando necessário, abrir a versão legada por seção.
